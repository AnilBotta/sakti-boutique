# 03 — Storefront Rules

## Homepage
- Cinematic hero with editorial typography and large imagery
- Featured collections (festive, new arrivals, best sellers)
- Women / Men / Kids entry points
- Lookbook-style storytelling sections
- Newsletter capture
- Trust strip (shipping, returns, secure payments)

## Navigation
- Responsive mega menu on desktop
- Premium mobile drawer with category accordion
- Sticky, accessible header
- Search entry visible at all breakpoints
- Cart and wishlist icons with live counts

## Category & Subcategory Pages
- Match the locked category tree exactly
- Faceted filters: price, size, color, fabric, occasion
- Sort: new, price asc/desc, best sellers
- Breadcrumbs always visible
- Collection banners at the top of each landing
- Stock-aware product cards with badges

## Product Detail Page (PDP)
- Premium image gallery with zoom / lightbox
- Variant selectors (size, color where applicable)
- Size guide modal
- Stock status and low-stock urgency
- Detailed description, fabric care, shipping & returns
- Wishlist toggle
- Related products + recently viewed
- Trust indicators near the CTA
- **Try Me / Virtual Try-On CTA** where applicable (see `05-tryon-rules.md`)

## Cart
- Slide-in cart drawer for quick actions
- Full cart page for review
- Promo code field
- Clear order summary with shipping estimate
- Smooth empty-cart and loading states

## Checkout
- Stripe-powered checkout
- Guest checkout supported
- Address form with validation and autocomplete-ready
- Mobile-optimized step flow
- Order confirmation with clear receipt

## Customer Account
- Login / signup
- Order history with status
- Saved addresses
- Wishlist
- Profile basics

## Social Proof
- Reviews and ratings on PDPs
- Photo reviews where possible
- Best Seller / New / Featured badges

> Mobile-first is mandatory: design every flow for the phone first, then enhance for desktop. Accessibility specifics live in `08-performance-seo-rules.md`.
