## Research priorities
When benchmarking, pull:
1. `<title>` and `<meta name="description">` from homepage, one category, one PDP
2. H1 / H2 structure
3. Visible JSON-LD blocks (`Product`, `BreadcrumbList`, `Organization`, `FAQPage`)
4. PDP description word count and section structure
5. Internal linking pattern from category → PDP → related
6. Any editorial/journal/lookbook content

Never copy wording verbatim — extract patterns and vocabulary only.
